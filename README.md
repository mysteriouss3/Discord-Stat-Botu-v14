### Kurulum:

1. Projeyi klonlayın:

```bash 
git clone https://github.com/mysteriouss3/Discord-Stat-Botu-v14.git
```
2. Proje icerisinde System.js doldurun.
```bash
Providers/Settings/System.js
```
3. Npm modullerini indirin.

```bash
npm install
```
4. Pm2 kurun.

```bash
npm install -g pm2
```

5. Botu çalıştırın.
```bash
pm2 start
```
